# Step 6: Imamge Capture and Analysis

### In this part we will show you how to set up your portable computer to take pictures from the webcam, send them to a Google Cloud Storage, and analyze them to detect if there is a dog. Alternatively, you could set up a bluetooth camera (connected to a desktop computer) as a webcam and fix it to the dispenser. This will simplify as you dont need the whole computer in in the physical presence of the dispenser in order for the script.
